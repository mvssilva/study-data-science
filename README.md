## 📚 Conteúdo do Curso

Nesta seção, documento o progresso inicial no curso de Ciência de Dados e Estatística, focando em lógica, probabilidade e manipulação de dados com Python.

* **Aula 01: Introdução e Regras do Jogo** Exploração da metodologia do curso e análise de segurança de dados através do estudo de padrões em senhas numéricas. Foco em entender "o que programar" antes de "como programar".
* **Aula 02: Fun with Probabilities** Introdução aos conceitos fundamentais de probabilidade e estatística. Uso de Python para simular eventos aleatórios e compreender distribuições de frequência.
* **Aula 03: Manipulação Básica de Dados** Primeiros passos com a biblioteca **Pandas**. Carregamento de datasets, filtragem, agregação de informações e conceitos iniciais de análise exploratória de dados.
